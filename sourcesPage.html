Sources

To create this project, I used a variety of sources attributed below. Each source is a link accompanied by a short description of how it was used. All conclusions and observations are entirely my own.

For information on cases, Oyez and the Supreme Court Website were used. Case information was gathered and synthesized into a Word document. Questions were written based on the facts of each case while Conclusions were written based on the conclusions found either on Oyez or through the Opinion of the Court.

For information on terms, Middle Tennessee State University's First Amendment Enclyopedia was used. Terms were written in my own words, based on the information from MTSU.

All other information was written either based on conclusions I drew from research, or fromt information learned in class.

Website was written in HTML, CSS, and JavaScript. A template from prowebdesign.ro was used as a basis for this project, with heavy modification to create more customized CSS and add project related customizations, including the timeline.
